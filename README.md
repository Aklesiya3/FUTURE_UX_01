# Future Interns Task 1 Dental Clinic Website Redesign – PureCare

## Project Overview

This project is part of UI/UX Design Task 1 (2026).
The objective was to redesign a local dental clinic website with a focus on improving lead generation and appointment bookings.

---

## Target Users

* Patients searching for dental services
* First-time visitors looking for trusted clinics
* Users who want quick and simple booking

---

## Layout Structure

The website follows a clear conversion-focused structure:

Homepage → Services → Contact → Book Appointment

The homepage highlights the value proposition and CTA.
Services are clearly categorized.
Contact and booking pages simplify user action.

---

## UX Decisions

* Strong and visible “Book Appointment” buttons
* Clear visual hierarchy using spacing and sections
* Step-by-step booking flow (Date → Time → Specialist)
* Trust elements (experience section, contact info, map)
* Mobile-friendly navigation

---

## How the Design Improves Conversions

* Reduces confusion with clear service presentation
* Guides users toward a single primary action
* Minimizes friction in the booking process
* Builds trust with visible credibility elements
